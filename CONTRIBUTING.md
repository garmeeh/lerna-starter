## Project Set Up

1. Fork & Clone repo
2. `npm install` to install your dependencies
3. `npm run test` to validate you got it working
4. Create a branch for your work
5. Open a PR

## Making Changes

- Please update the documentation if needed
- Tests where possible should be added
